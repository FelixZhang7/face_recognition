⚙️ 使用方法
--------------------

- 下载 mnist 数据
```bashrc
$ wget https://pjreddie.com/media/files/mnist.tar.gz
$ tar xvzf mnist.tar.gz
```
- 编译 example 程序
```bashrc
$ make example01
$ ./example01 images/sample.png 3
```
